A machine learning-based web application that predicts student academic performance. This full-stack project features a Streamlit frontend and a Flask API backend, providing an interactive interface for educators and students to forecast grades based on various academic and demographic factors.


## Installation

1-Clone the repository:

```bash
git clone https://github.com/mohamed1-abdeldayem/StudentPerformacne.git
cd StudentPerformacne
```


2-Install required dependencies:

```bash
pip install -r requirements.txt
```

```bash
cd API
pip install -r requirements.txt
```


3-Start the Flask API Server (Backend):

```bash
cd API
python app.py 
```

4-Start the Streamlit App (Frontend):

```bash
streamlit run student_performance_app.py
```
